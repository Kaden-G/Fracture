# FRACTURE

A single-file browser strategy game. Four factions fight for the broken grid of Nexus —
control 3 of 5 Core Nodes to win. Dominate, negotiate, betray, survive.

**Play:** https://kaden-g.github.io/Fracture/

## Features
- 1–4 players (any mix of humans + AI) on one device — pass-and-play hot-seat
- Diplomacy: non-aggression pacts, betrayal grudges, anti-snowball coalition
- Node synergies, asymmetric faction perks, choice events
- Online cross-device play (Firebase) — *in progress*

Everything lives in `index.html`. No build step.
